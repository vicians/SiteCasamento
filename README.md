> **"Nós amamos porque Ele nos amou primeiro." (1 João 4:19)**

Este é o repositório oficial do site de casamento de **Samuel & Gabriela**. O projeto foi concebido para ser mais do que um convite digital: é um sistema completo de gestão de convidados, lista de presentes e experiência interativa, desenvolvido com foco em performance e design minimalista.

---

## 🚀 Tecnologias Utilizadas

O projeto utiliza uma stack moderna de "Backend-as-a-Service" para garantir agilidade e escalabilidade:

- **Frontend:** HTML5, CSS3 (Custom Properties) e Vanilla JavaScript.
- **Banco de Dados:** [Supabase](https://supabase.com/) (PostgreSQL) para Lista de Presentes.
- **Hospedagem:** [Vercel](https://vercel.com/) com deploy contínuo via GitHub.
- **Segurança:** Row Level Security (RLS) configurado no banco para proteção de dados.

---

## ✨ Funcionalidades

- [x] **Contagem Regressiva:** Timer em tempo real para o dia 20 de novembro de 2026.
- [x] **Lista de Presentes Dinâmica:** Integração com banco de dados para exibição de itens e valores.
- [x] **Sistema de RSVP:** Confirmação de presença enviando dados diretamente para o Supabase.
- [x] **Design Responsivo:** Adaptado para mobile, priorizando o uso por convidados via WhatsApp.
- [x] **Efeitos de UI:** Navegação inteligente (Smart Navbar) e revelação de conteúdo ao scroll.
